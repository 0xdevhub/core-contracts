# 0xdevhub - core contracts

Available scripts in the `package.json`:

```shell
pnpm compile
pnpm typechain
pnpm test
pnpm test:watch
pnpm test:coverage
```

Try running some of the following tasks:

```shell
dlx hardhat help
dlx hardhat test
REPORT_GAS=true dlx hardhat test
dlx hardhat node
dlx hardhat run scripts/deploy.ts
```
