```shell
pnpm compile
pnpm typechain
```

## Available scrips by `dlx hardhat`

```shell
dlx hardhat help
dlx hardhat test
REPORT_GAS=true dlx hardhat test
dlx hardhat node
dlx hardhat run scripts/deploy.ts
```

## Testing

```bash
pnpm test
pnpm test:watch
pnpm test:coverage
```
